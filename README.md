# VLESS to OpenVPN Adapter

A lightweight bridge that converts VLESS proxy nodes into a standard OpenVPN server. Perfect for using VLESS proxies on devices that only support OpenVPN, or for sharing a single VLESS connection across your local network.

## 🚀 Features
- **Auto-Subscription**: Import VLESS/VMess/Trojan nodes directly from a URL.
- **Sing-box Core**: High-performance proxying using the modern Sing-box engine.
- **Transparent Bridge**: Automatically handles routing, DNS hijacking, and NAT.
- **Easy Setup**: Generates a ready-to-use `.ovpn` file for your clients.

---

## 🛠 Prerequisites

- **Linux**: `openvpn`, `iptables`, and `sing-box` installed in your PATH.
- **Windows/macOS (Client only)**: An OpenVPN client app.

---

## 🏃 How to Run

1. **Build or Download** the `adapter` binary.
2. **Start the adapter** (requires sudo/admin for networking):
   ```bash
   # Standard run (silent logs)
   sudo ./adapter -sub "YOUR_SUBSCRIPTION_URL"

   # Verbose run (show all proxy/vpn logs)
   sudo ./adapter -sub "YOUR_SUBSCRIPTION_URL" -v

   # If you see "legacy DNS servers" errors, use this hack:
   sudo ENABLE_DEPRECATED_LEGACY_DNS_SERVERS=true sh -c './adapter -sub "YOUR_SUBSCRIPTION_URL"'
   ```
3. **Connect!** Use the generated `client.ovpn` file with your favorite VPN client.

## 🖥 Client Setup Guide (GUI)

### 🐧 Linux (Gnome)
1. Open **Settings** -> **Network**.
2. Click the **+** icon next to VPN.
3. Select **Import from file...**.
4. Choose the `client.ovpn` file generated by the adapter.
5. Click **Add**, then toggle the switch to connect!

### 🐧 Linux (KDE Plasma)
1. Click the **Network** icon in the system tray.
2. Click the **Configure** (gear) icon.
3. Click **+ Add New Connection**.
4. Scroll down to **Import VPN connection...** and select the `client.ovpn` file.
5. Click **Create** and then connect.

### 🪟 Windows
1. Download and install **[OpenVPN GUI](https://openvpn.net/community-downloads/)**.
2. Right-click the OpenVPN icon in the system tray (bottom right).
3. Select **Import** -> **Import file...**.
4. Select the `client.ovpn` file.
5. Right-click the icon again and select **Connect**.
6. **Note**: If you are connecting to an adapter running on another Linux machine, ensure you used the `-host` flag with the Linux machine's local IP!

---

## ⚙️ Advanced Usage

- **Expose to Network**: If you want other devices in your house to connect, use your local IP:
  ```bash
  sudo ./adapter -sub "..." -host 192.168.1.10
  ```
- **Node Selection**: Currently, the adapter uses the first valid node in the subscription. Support for node selection and load-balancing is coming soon!

---

## 🔒 Security Note
This tool uses OpenVPN's **Static Key** mode for maximum simplicity and performance in local environments. For use over public, untrusted networks, ensure your underlying VLESS connection is encrypted (TLS/Reality).
